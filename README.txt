Архивът съдържа 3 папки - fonts, js и images:

-Във първата е поместен съответният използван шрифт в приложението.
-В js са поместени използваните библиотеки и JavaScript файлове.
-В images са поставени изображенията на съответните лекарства.

В архивът има също два HTML документа: 

- 18_61534_61488_html.html - чрез него се стартира приложението, като може да се използва всеки 
  съвременен браузър (Mozilla Firefox е за предпочитане);

- 18_61534_61488_description_html.html - в него са поместени описанията и снимките на всеки от продуктите;

Два XML документа: 

- 18_61534_61488_xml.xml - съдържа всички лекарства и техни характеристики;

- 18_61534_61488_description_xml.xml - съдържа описание и снимки на всички лекарства 
  (използва се заедно с 18_61534_61488_description_html.html)

Два XSLT документа: 

- 18_61534_61488_xslt.xsl - с негова помощ, съдържанието на XML файлът се визуализира в браузъра;

- 18_61534_61488_description_xslt.xsl - работи заедно с 18_61534_61488_description_xml.xml и  
  18_61534_61488_description_html.html ;

Един CSS документ:

- 18_61534_61488_css.css  - спомага за козметичните характеристики на приложението, използва се заедно с 
  18_61534_61488_html.html ;

Един PDF документ: 

- 18_61534_61488_ Documentation.pdf - това е документацията на приложението, в нея може да се намери 
  описание, ръководство, списък с използваните технологии и т.н. ;
